## 태그 기능을 만들고있다.

기존 블로그들의 카테고리가 아닌 해시태그의 느낌으로 게시글의 성격을 여러 태그들로 나타내려고 한다.

## 구현방식

> 1. 게시글을 작성한 mdx의 front-matter에 tag 필드를 추가하여 태그들을 작성한다.
> 2. 모든 포스트 tag 필드의 문자열을 가져와 띄어쓰기로 분리한다.
> 3. 반복문으로 검사하여 처음 나온 태그는 오브젝트에 추가한다.
> 4. 이미 나왔던 태그는 개수를 카운팅해준다.

결과적으로 나온 예시는 다음과 같다.

```typescript
/* 생성 */
let tags: { [k: string]: number } = {};
postsList.forEach((post) => {
  post.tag.split(' ').forEach((tag) => {
    if (!Object.keys(tags).includes(tag)) tags[tag] = 0;
    tags[tag]++;
  });
});

/* 결과 */

tags = {
  "코딩": 1,
  "일상": 1,
  "고정": 2,
}
```
